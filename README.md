# Projeto-Xadrez-JS ♟️

Jogo de xadrez completo desenvolvido com HTML5, CSS3 e JavaScript puro, sem uso de bibliotecas externas.

## 🚀 Teste agora
**[Clique aqui para jogar](https://erikalvesdasilva12-dev.github.io/Projeto-Xadrez-JS/)** 


## 🎯 Funcionalidades
- Tabuleiro 8x8 com design responsivo
- Movimentação completa de todas as peças
- Validação de jogadas seguindo regras oficiais
- Detecção de xeque e xeque-mate
- Reiniciar partida

## 🛠️ Tecnologias Utilizadas
- `HTML5` para estrutura
- `CSS3` para estilização e layout
- `JavaScript` para toda a lógica do jogo

## 💻 Como rodar localmente
1. Baixe o projeto: `Code` > `Download ZIP`
2. Extraia a pasta
3. Abra o arquivo `index.html` no seu navegador
